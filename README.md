## 📌 Overview

Lapor-in is a full-stack public complaint management platform designed to simplify communication between citizens and administrators.

Users can submit reports, monitor complaint progress, and receive status updates through an intuitive interface, while administrators can efficiently review and manage incoming reports.

---

## ✨ Features

### 👥 User

- Register & Login
- Submit Complaint Reports
- Upload Report Images
- Track Complaint Status
- View Complaint History
- Manage Profile

### 🛡️ Admin

- Secure Admin Authentication
- Dashboard Overview
- Manage Complaint Reports
- Update Report Status
- Manage Users

### ⚡ System

- JWT Authentication
- REST API
- Responsive UI
- MySQL Database
- Clean Dashboard Interface

## 🚀 Installation

Clone this repository

```bash
git clone https://github.com/raxhasnoiq/Lapor-in.git
````

Go to project folder

```bash
cd Lapor-in
```

Install frontend dependencies

```bash
npm install
```

Install backend dependencies

```bash
cd backend
npm install
```

Start backend server

```bash
npm run dev
```

Start frontend

```bash
cd ..
npm run dev
```
